# Outlook Invoice Automation

Automation project focused on extracting, classifying and processing invoice PDF files received through Outlook mailboxes.

---

## Business Problem

Many finance and administrative departments still process supplier invoices manually:

- Downloading attachments one by one
- Renaming PDF files manually
- Classifying invoices into folders
- Searching for supplier information
- Validating invoice data manually

This creates operational bottlenecks, repetitive work and high risk of human error.

---

## Solution

Python-based automation workflow for:

- Outlook email processing
- PDF attachment extraction
- Invoice classification
- Duplicate detection
- OCR-based document reading
- Automated file organization
- Validation support processes

---

## Technologies Used

- Python
- Outlook COM Automation
- pandas
- pdfplumber
- OCR / Tesseract
- File System Automation

---

## Key Features

- Automatic Outlook mailbox scanning
- Invoice PDF download automation
- Supplier-based classification
- Duplicate invoice detection
- OCR support for scanned invoices
- Automated folder structure generation
- Logging and validation reporting

---

## Business Impact

Typical operational improvements:

- Reduction of repetitive manual work
- Faster invoice processing
- Better document traceability
- Reduced risk of duplicated invoices
- Improved operational efficiency
