# ERP Automation Portfolio

Portfolio focused on ERP automation, data migration validation, financial data reconciliation and business process optimization using SQL, Python and AI.

---

## About

This repository contains business-oriented automation and data validation projects inspired by real ERP transformation and migration scenarios.

Main focus areas:

- ERP Migration Validation
- NAV → SAP Data Controls
- Financial Data Reconciliation
- SQL Data Analysis
- Python Process Automation
- Document Automation
- Business Process Optimization
- AI-Powered Operational Efficiency

---

## Portfolio Projects

### 1. Outlook Invoice Automation
Automation for invoice extraction, PDF classification and document processing from Outlook mailboxes.

### 2. NAV → SAP Validation Toolkit
Validation and reconciliation toolkit for ERP migration scenarios between Microsoft Dynamics NAV and SAP S/4HANA.

### 3. Financial Reconciliation Engine
Automated reconciliation and matching processes for financial and operational datasets.

### 4. ERP Data Quality Checks
Scripts and controls focused on detecting inconsistencies and improving ERP data quality.

The goal of this portfolio is to demonstrate practical automation, ERP validation and business process optimization capabilities applied to real operational environments.
